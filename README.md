# DK Research Log

Personal research portfolio and writing — [dkresearchlog.com](https://dkresearchlog.com)

## Deploy

1. Push this repo to GitHub
2. Import project on [vercel.com](https://vercel.com)
3. Add custom domain `dkresearchlog.com` in Vercel dashboard
4. In Namecheap DNS, set nameservers to Vercel's (provided in dashboard)

## Structure

Single `index.html` — all pages, styles, and logic self-contained.
Add blog post content by editing the `posts` array in the script section.
